Deprecated

Note that this style is deprecated and will be removed in a future version of AUI - you should only use this if you have a critical requirement to maintain the striping style in the short term.

To add zebra striping (and remove the borders) add the class aui-zebra to the table.

Test note: striping should start on the second row in both tables.
