Toolbar

Demonstrates a toolbar with the various button types and states.

Notes

Button types:

  • Standard/default
  • Primary - a button with an elevated importance, eg. the "publish" button in an editor. Note: there should only be one primary button per toolbar, otherwise this is meaningless.
  • Dropdown - a button with a dropdown icon, that opens a dropdown menu (one action)
  • Split button - a button with a dropdown icon, where the button performs an action but the dropdown icon opens a menu (two different actions). eg. the "apply colour" button in an editor, which also has a dropdown colour selector.

Button states:

  • Default
  • Active/on - eg. includes buttons which toggle a state, such as the "format: bold" button in an editor
  • Disabled - for buttons which are present/visible but not currently usable
  • Hover/focus

All buttons may include text, an icon or both (some buttons may include a second icon to indicate a dropdown). For icon-only buttons the text is hidden in the icon element, so it's still available to screen readers.
