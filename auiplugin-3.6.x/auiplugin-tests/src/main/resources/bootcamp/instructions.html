AUI Bootcamp Exercise

Setup

  1. It is assumed that your dev environment is already set up with Maven etc and you have a suitable text editor.
  2. (NB: do not commit your bootcamp exercise files! ;))
  3. If you're reading these instructions it should mean you've checked out https://studio.atlassian.com/svn/AJS/trunk but if not please check out AUI now.
  4. go to the root of your AUI checkout and run mvn install -Dmaven.test.skip
  5. go to auiplugin-test
  6. mvn refapp:run (this may take a little while first go)
  7. While the refapp is startup up... on your file system, go to auiplugin-tests\src\main\resources\bootcamp ...note this location is not significant, it's simply convenient for loading resources for the bootcamp exercise.
  8. For this exercise, you will be editing bootcamp.html and bootcamp.js so open them in your choice of editor.
  9. open bootcamp.html in browser
  10. open http://docs.atlassian.com/aui/latest/sandbox/
  11. If in doubt, check your implementation against bootcamp-example.html

Exercise

  1. Tabs
    • Using sandbox, create tabs including one named "form", one named "dialog".
    • Give the tabs-pane divs the IDs bootcamp-form and bootcamp-dialog; then update the links inside tabs-menu
    • Remove the unused tab controls and tabs.
    • Change the tabs to be vertical tabs (hint: check the class names in the documentation or view the full Tabs documentation page).
    • Copy the tabs HTML and Javascript into bootcamp.html/.js then reload bootcamp.html in your browser.
  2. Dialog
    • Repeat the process to add a dialog button inside the dialog tab.
  3. Form
    • Repeat the process to add a form inside the form tab.
  4. Freestyle! Add a table, or a message, or a toolbar...
